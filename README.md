## 실행방법
1. 현재 리포지토리를 클론한다.
   - `git clone https://github.com/mandariny/wanted-pre-onboarding-backend.git`
2. MySQL을 구동한다.
   - `docker-compose up -d`
   - sample data가 삽입된 mysql이 실행된다.
3. SpringBoot를 구동한다.
